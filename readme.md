## Installations Steps

- Copy `.env.example` into `.env` (See [Env Example File](#env-example-file))
- `composer install`
- `composer dump-autoload`
- `php artisan key:generate`
- `php artisan migrate:fresh --seed`
- `php artisan passport:install --force`
- `php artisan serve`
